Home

fly is a fast and simple mirco-framework for small web-applications. It offers request dispatching (Routes) with url parameter support, Templates, key/value Databases, a build-in HTTP Serverand adapters for many third party WSGI/HTTP-server and template engines. All in a single file and with no dependencies

Installation and Dependencies

You can install fly with or just download fly.py and place it in your project directory.
